Récupérer les IDs d'observations iNaturalist par identifiant OSM

Comment ça marche ?

Cette page permet de récupérer les identifiants des observations naturalistes (iNaturalist) associées à un lieu précis sur OpenStreetMap (OSM).

Comment faire ?

  1. Copiez l'URL d'un nœud OSM (par exemple, un point d'intérêt sur OpenStreetMap).
  2. Collez cette URL dans le champ ci-dessus et cliquez sur "Récupérer les IDs".
  3. Les identifiants des observations iNaturalist liées à ce lieu s'affichent.
  4. Utilisez le bouton "Copier les IDs" pour les copier dans votre presse-papiers.

À quoi ça sert ?
Cela permet de retrouver facilement toutes les observations naturalistes (plantes, animaux, etc.) enregistrées sur iNaturalist pour un lieu spécifique, comme un jardin, un parc ou un point d'intérêt naturel.
